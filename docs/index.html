MCraft

El bot ideal para tu servidor de discord y para tu servidor de minecraft.

¿Que funciones cumple el bot?

MCraft esta equipado con funciones especiales para owners de servidores de minecraft que quieren integrar su comunidad a servidores de discord.

Icon

Verificacion "In-Game"

Es un sistema creado para que al entrar al servidor discord tengas que verificarte mediante un codigo que se te brindara ingame.

Icon

Integracion con el chat y consola

Con esta funcion integramos el chat de discord con el del juego, haciendo posible ver los mensajes del juego en el discord y viceversa.Tambien permite a los admins leer la consola.

Icon

Sistema de soporte de tickets

Nunca antes se le facilito tanto al staff leer reportes, con este sistema de tickets el usuario podra crear un canal que solo lo veran el y el staff competente mediante un sistema intuitivo y grafico.

Icon

Aumento de estadisticas

Esta comprobado que mantener una comunidad activa en una aplicacion como discord influye directamente en la cantidad de jugadores y en su estadia en el servidor.

Icon

Sanciones

Puedes hacer cumplir las normas mediante las sanciones en discord tales como silenciar, hechar, expulsar temporalmente y permanente, hacer advertencias y todo esto de una forma muy facil.

Icon

Contacto directo con el soporte del bot

¿No entendes alguna funcion?, ¿Queres sugerir algo?, ¿Encontraste un bug?, MCraft es de los pocos bots que permite a los usuarios contactarse directamente con el desarrollador del mismo.

¿Aun no cumplimos tus espectativas?

Te aseguramos que nuestro plugin no causara problemas en tu servidor y contamos con soporte para CraftBukkit, Spigot y Paper.
